## Math

### ceil

大于等于 x，并且与它最接近的整数

### floor

小于等于 x，且与 x 最接近的整数

### round

4舍5入的计算，入的时候是到大于它的整数

```java
// -11
System.out.println("Ceil:"+Math.ceil(-11.5));
// 12
System.out.println("Ceil:"+Math.ceil(11.5));
// -12
System.out.println("floor:"+Math.floor(-11.5));
// 11
System.out.println("floor:"+Math.floor(11.5));
// -11
System.out.println("round:"+Math.round(-11.5));
// 12
System.out.println("round:"+Math.round(11.5));
```
