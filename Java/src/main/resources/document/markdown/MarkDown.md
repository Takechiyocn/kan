### 表格

对齐方式|左对齐(默认)|中间对齐|右对齐
---|---|:---:|---:
id1|1|a|内容右对齐1
id2|12|ab|内容右对齐12
id3|123|abc|内容右对齐123
id4|1234|abcd

* 第二行分割表头和内容
* `-` 有一个就行，为了对齐，多加了几个
  * 文字默认居左
  * `-`两边加`:`表示文字居中
  * `-`右边加`:`表示文字居右
  * 原生的语法两边都要用`|`包起来,此处省略
